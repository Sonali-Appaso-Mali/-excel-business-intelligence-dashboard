# 📊 Interactive Sales Dashboard in Excel

## 🔍 Overview
An interactive Sales Dashboard built in Microsoft Excel to analyze and visualize business sales performance from 2021–2024.

This dashboard provides insights into sales trends, profit growth, customer performance, and state-wise analysis using interactive visualizations and slicers.

---

## 🎯 Objectives
- Analyze sales performance across multiple categories
- Monitor yearly profit growth and trends
- Identify top-performing products and customers
- Visualize monthly and state-wise sales distribution
- Create an interactive and user-friendly business dashboard

---

## 📈 Dashboard Features
✔️ Sales by Category Analysis  
✔️ Profit Trend Over Time  
✔️ State-wise Sales Visualization  
✔️ Customer Count Analysis  
✔️ Top 5 Customers by Profit  
✔️ Monthly Sales Trend Analysis  
✔️ Interactive Slicers for Dynamic Filtering  

---

## 🛠️ Tools & Technologies Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Data Visualization

---

## 💡 Key Insights
- Monitors and Pens emerged as top-selling products
- Technology category generated the highest profit growth
- Sales are concentrated in selected high-performing states
- Top customers contributed significantly to overall profit
- Monthly sales peaked around Month 8
